# Linksaw Snippets

Type less. Do more. Linksaw Snippets is a global text autocomplete and expansion extension for Chrome. 

Create snippets for your most-used phrases and expand them instantly anywhere you type.

## Features

- **⚡ Lightning Fast:** Just type your trigger key and your predefined phrase expands instantly inside any text field in your browser.
- **☁️ Cloud Sync:** Sync your shortcuts securely across all your devices using Google Sign-In authentication.
- **⚙️ Customizable:** Define your own trigger keys, create snippets with or without titles, and effortlessly manage your library.

## Installation

Visit the [Chrome Web Store](https://chrome.google.com/webstore) to install Linksaw Snippets for free.
