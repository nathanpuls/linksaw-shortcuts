# Linksaw

**Tiny Markdown pages. Big sharing energy.**

Linksaw is a simple way to turn plain Markdown into clean, readable web pages. No dashboard labyrinth. No design decisions. Just write, save, and share.

[Get Started](#get-started) · [See Examples](#markdown-showcase)

---

## Why Linksaw?

Most publishing tools feel heavier than the thing you’re trying to publish.

Linksaw is for quick pages like:

- AI-generated notes
- Simple landing pages
- Link lists
- Recipes
- Class handouts
- Client reports
- Tiny personal websites

> If it can be written in Markdown, it can become a page.
